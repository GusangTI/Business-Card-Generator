SOBRE NÓS

Este Gerador de Cartão de Visita é um projeto prático desenvolvido por mim (Gustavo de Oliveira Garcia Ribas) para a Atividade Somativa 1 (AS1) da disciplina de Fundamentos de Programação Web.

O principal objetivo foi demonstrar os conhecimentos fundamentais da programação web, incluindo:

HTML: Para estruturar o conteúdo das páginas.

CSS: Para estilizar a aplicação, criando o layout da barra lateral e o design dos cartões.

JavaScript: Para adicionar interatividade, validar os formulários e transferir os dados entre as páginas usando o localStorage.
